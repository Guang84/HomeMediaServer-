About the Project

The Media Server Application is designed to help families and friends share selected media content without exposing all data to everyone. Developed primarily in response to internet restrictions in Manipur, this project simplifies file sharing within a local network and keeps track of important moments even when offline.

By using this media server, you no longer need to repeatedly transfer data from one device to another. Once files are added to the server, they are accessible from any device on the network, making content management effortless.

User Guide

Getting Started

  1. Clone the Repository: Clone the project repository from GitHub using the following command: 
    git clone https://github.com/Guang84/HomeMedia_Server.git
  2. Install Dependencies: Navigate to the project directory and install the necessary Python packages: 
    pip install -r requirements.txt
  3. Run the Application: Start the Flask development server with: 
    python3 app.py
    For home network access, use: 
    open link in server

Accessing the Application

Open your browser and go to the local IP address assigned to the hosting device (e.g., http://192.168.1.10:5000/) to access the media server from any device on your home network.

Features

  • Media Sharing: Seamless sharing of videos, images, and files across devices on the same network.
  • No User Authentication: Simple and secure access without the need for user accounts.
  • Dynamic Pagination: Browse through large directories effortlessly.
  • Download Support: Download media files or entire folders as ZIP archives.
  • Offline-Friendly: Operates smoothly without internet access.
  • Responsive Design: Compatible with desktops, tablets, and smartphones.
